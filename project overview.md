# Project Overview

## Project Title
Advanced NLP Sentiment Analysis System

## Problem Statement
Businesses receive thousands of customer reviews daily. 
Manually analyzing sentiment is time-consuming.

## Objective
Build a deep learning NLP system that automatically predicts customer sentiment from text reviews.

## Technologies Used
- Python
- TensorFlow
- FastAPI
- Docker
- Scikit-learn
- Monitoring & Logging

## Features
- Text preprocessing
- LSTM deep learning model
- Real-time API prediction
- Docker deployment
- Monitoring system
